Wat is machine learning?

Nu één van de snelst-groeiende markten ter wereld, het is de technologie waarop bijna heel de toekomst op zal rusten. Het staat zelf op het idee dat een programma zelf kan “leren”, en er zijn eindeloos veel toepassingen. Ingenieurs in Tesla werken aan zelfrijdende auto’s met machine learning. Programmeurs in Apple werken aan gezichtsherkenning technologie met machine learning. Zelfs aandelen worden verhandeld door bots getraind met machine learning. Machine learning is de wetenschap van computer algoritmes die beter worden door middel van ervaring. Er zijn talloze machine learning algoritmes die allemaal voor bepaalde toepassingen worden gebruikt, van soorten die een programma een omgeving beter leert kennen tot soorten die data probeert te begrijpen en daarmee toekomstige gebeurtenissen voorspellen.
