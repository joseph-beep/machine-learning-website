Titel 3

Machine learning is de wetenschap van computer algoritmes die beter worden door middel van ervaring. Er zijn talloze machine learning algoritmes die allemaal voor bepaalde toepassingen worden gebruikt, van soorten die een programma een omgeving beter leert kennen tot soorten die data probeert te begrijpen en daarmee toekomstige gebeurtenissen voorspellen.
